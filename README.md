# Bootstrap 3 - Starter Site

1. Clone or download this repo in to the root of your project directory.
2. Run `npm install` (or `sudo npm install` for Linux & OSX users) in the terminal from the root of your project.
3. Run the command `grunt` or `grunt default` to compiles the bootstrap.
4. All done!

If you want to have grunt watch your files for changes and automatically recompile bootstrap and live reload your webpage, run `grunt watch` in the terminal from the root of your project.

Please NOTE: You must have a live reload plugin installed in either Google Chome or Firefox for live reload to work. Alternatively add this line to you project:

`<script src="http://localhost:35729/livereload.js"></script>`
